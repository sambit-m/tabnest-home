# TabNest Privacy Policy

Effective date: February 21, 2026

TabNest is a Chrome extension that helps users organize open tabs by host, search tabs, and create native Chrome tab groups.

## Data We Access

TabNest uses Chrome extension permissions to access:
- Open tab metadata (such as tab URL, title, tab ID, and window ID)
- Tab group operations (creating/updating Chrome native tab groups)

This may include a user's web history context (open/visited pages visible in tabs).

## How Data Is Used

Data is used only to provide TabNest features:
- Group tabs by host
- Search tabs by host/title
- Select, activate, close, and group tabs

## Data Sharing and Selling

TabNest does not sell, transfer, or share user data with third parties.

## Data Storage

TabNest stores only local extension preferences needed for functionality (for example, selected theme), using browser local storage.
Tab/tab group data is processed locally in the browser at runtime.

## Remote Code

TabNest does not use remote code. All executable code is packaged with the extension.
